# Agent memory

Nabu treats markdown files as the durable knowledge layer.

That means a human can browse the same knowledge space that an agent reads from disk.

## Why this matters

- The notes stay portable.
- The UI stays optional.
- The system remains inspectable.
- AI agents can work from the same source of truth.
